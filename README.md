# TenaMeal

An AI-powered Hospital Nutrition & Meal Management System developed during my internship at the Ministry of Innovation and Technology (MINT) in Ethiopia.

## Features
- Doctor-approved meals
- Allergy and dietary restriction management
- Meal requests and history
- Notifications
- AI-assisted development

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Lovable
